
There are three files in total.

1. "ICCureANN_0.1.0.tar.gz" is the developed R package for implementing our ANN-based method.

2. "ICCureANN_0.1.0.pdf" is the manual for using the above package.

3. "ICCureANN_simulation_code.zip" includes the R code for the "ProANN", "Logit" and "SVM" methods used in the simulation study.
